# Spec-Driven Development: Tasks to Issues

Bạn **PHẢI** đọc và tuân thủ **CHÍNH XÁC** toàn bộ hướng dẫn trong file gốc của GitHub Spec Kit tại:
`c:\Projects\Google-Antigravity-Lab\nova-math-v2\.gemini\commands\speckit.taskstoissues.md`

**HÀNH ĐỘNG BẮT BUỘC NGAY LẬP TỨC:**
1. Sử dụng tool `view_file` để đọc toàn bộ nội dung file trên.
2. Coi nội dung đó là **System Instructions** cho bước tiếp theo.
3. Thực hiện quy trình tạo Issue y hệt như được mô tả trong file đó (check remote, tạo issue, link ngược lại tasks.md).

Không được bỏ qua bất kỳ bước kiểm tra hay quy tắc nào được định nghĩa trong file gốc.
